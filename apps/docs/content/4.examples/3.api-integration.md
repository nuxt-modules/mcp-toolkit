## Overview

This page shows a simple example of how to integrate external APIs with MCP tools.

## Simple API Example

Here's a basic example of calling an external API:

```typescript [server/mcp/tools/api-example.ts]
import { z } from 'zod'

export default defineMcpTool({
  name: 'fetch-data',
  description: 'Fetch data from an external API',
  inputSchema: {
    url: z.string().url().describe('API endpoint URL'),
  },
  handler: async ({ url }) => {
    try {
      const response = await fetch(url)

      if (!response.ok) {
        throw new Error(`API error: ${response.statusText}`)
      }

      const data = await response.json()

      return {
        content: [{
          type: 'text',
          text: JSON.stringify(data, null, 2),
        }],
      }
    }
    catch (error) {
      return {
        content: [{
          type: 'text',
          text: `Error: ${error instanceof Error ? error.message : String(error)}`,
        }],
        isError: true,
      }
    }
  },
})
```

## Best Practices

1. **Use environment variables**: Store API keys in environment variables
2. **Handle errors**: Always handle API errors gracefully
3. **Validate responses**: Validate API responses before returning them
4. **Security**: Never expose sensitive credentials or allow arbitrary API calls without proper validation
