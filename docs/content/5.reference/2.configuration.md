## Module Configuration

Configure the module in your `nuxt.config.ts`:

```typescript [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@hrcd/mcp'],
  mcp: {
    // Configuration options
  },
})
```

## Configuration Options

::field-group
  ::field{name="enabled" type="boolean"}
    Default: `true`
    
    Enable or disable the MCP server. When disabled, no MCP endpoints are created.
  ::

  ::field{name="route" type="string"}
    Default: `'/mcp'`
    
    The HTTP route where the MCP server will be accessible. This is the endpoint that MCP clients will connect to. The route should start with `/` and not conflict with other routes in your application.
  ::

  ::field{name="browserRedirect" type="string"}
    Default: `'/'`
    
    URL to redirect browsers when they access the MCP endpoint. When someone accesses the MCP endpoint in a browser (which sends `text/html` in the Accept header), they'll be redirected to this URL.
  ::

  ::field{name="name" type="string"}
    Default: `''`
    
    The name of your MCP server. This is used in the MCP protocol handshake.
  ::

  ::field{name="version" type="string"}
    Default: `'1.0.0'`
    
    The version of your MCP server. Follow semantic versioning (major.minor.patch).
  ::

  ::field{name="dir" type="string"}
    Default: `'mcp'`
    
    Base directory for MCP definitions (relative to `server/`). The module expects subdirectories:
    - `{dir}/tools/` - Tool definitions
    - `{dir}/resources/` - Resource definitions
    - `{dir}/prompts/` - Prompt definitions
  ::
::

## Default Configuration

If no configuration is provided, the module uses:

```typescript
{
  enabled: true,
  route: '/mcp',
  browserRedirect: '/',
  name: '',
  version: '1.0.0',
  dir: 'mcp',
}
```

## Runtime Configuration

Access configuration at runtime:

```typescript [server/api/config.ts]
export default defineEventHandler((event) => {
  const config = useRuntimeConfig(event).mcp

  return {
    enabled: config.enabled,
    route: config.route,
    name: config.name,
    version: config.version,
  }
})
```

## Complete Example

```typescript [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@hrcd/mcp'],
  mcp: {
    enabled: true,
    route: '/mcp',
    browserRedirect: '/docs',
    name: 'My MCP Server',
    version: '1.0.0',
    dir: 'mcp',
  },
})
```
