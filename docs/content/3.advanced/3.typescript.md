## Type Safety

The Nuxt MCP module provides full TypeScript support with complete type inference and type safety.

## Auto-Imports

All helper functions are automatically imported in your server files:

```typescript
// No imports needed - auto-imported!
export default defineMcpTool({
  name: 'example',
  inputSchema: {
    message: z.string(),
  },
  handler: async ({ message }) => {
    // message is typed as string
  },
})
```

Available auto-imports:
- `defineMcpTool`
- `defineMcpResource`
- `defineMcpPrompt`
- `defineMcpHandler`

## Type Inference

### Tool Input Types

Input types are automatically inferred from your `inputSchema`:

```typescript
export default defineMcpTool({
  name: 'example',
  inputSchema: {
    name: z.string(),
    age: z.number(),
    email: z.string().email().optional(),
  },
  handler: async ({ name, age, email }) => {
    // name: string
    // age: number
    // email: string | undefined
  },
})
```

### Tool Output Types

Output types are inferred from `outputSchema`:

```typescript
export default defineMcpTool({
  name: 'example',
  inputSchema: {
    value: z.number(),
  },
  outputSchema: {
    result: z.number(),
    doubled: z.number(),
  },
  handler: async ({ value }) => {
    const result = value * 2

    return {
      content: [{ type: 'text', text: String(result) }],
      structuredContent: {
        result,      // TypeScript knows this is number
        doubled: result * 2, // TypeScript knows this is number
      },
    }
  },
})
```

### Prompt Argument Types

Prompt argument types are inferred from `argsSchema`:

```typescript
export default defineMcpPrompt({
  name: 'example',
  argsSchema: {
    text: z.string(),
    maxLength: z.string().optional(),
  },
  handler: async ({ text, maxLength }) => {
    // text: string
    // maxLength: string | undefined
  },
})
```

## Type Definitions

Import types from `@hrcd/mcp`:

```typescript
import type {
  McpToolDefinition,
  McpResourceDefinition,
  McpPromptDefinition,
  McpHandlerOptions,
} from '@hrcd/mcp'
```

### Tool Types

```typescript
import type { McpToolDefinition } from '@hrcd/mcp'

const tool: McpToolDefinition = {
  name: 'example',
  inputSchema: { ... },
  handler: async (args) => { ... },
}
```

### Resource Types

```typescript
import type { McpResourceDefinition } from '@hrcd/mcp'

const resource: McpResourceDefinition = {
  name: 'example',
  uri: 'file:///example.txt',
  handler: async (uri) => { ... },
}
```

### Prompt Types

```typescript
import type { McpPromptDefinition } from '@hrcd/mcp'

const prompt: McpPromptDefinition = {
  name: 'example',
  handler: async () => { ... },
}
```

### Handler Types

```typescript
import type { McpHandlerOptions } from '@hrcd/mcp'

const handler: McpHandlerOptions = {
  name: 'example',
  tools: [ ... ],
  resources: [ ... ],
  prompts: [ ... ],
}
```

## Runtime Config Types

Access typed runtime configuration:

```typescript [server/api/config.ts]
export default defineEventHandler((event) => {
  const config = useRuntimeConfig(event).mcp

  // config is typed with:
  // - enabled: boolean
  // - route: string
  // - browserRedirect: string
  // - name: string
  // - version: string
  // - dir: string

  return config
})
```
